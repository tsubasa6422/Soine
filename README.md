# Soine  (そいね)

## サイト概要
### サイトテーマ
子育て情報の発信をおこなったり、地域の子育て世代向けのイベントなどの情報収集ができるSNSサイト

​
### テーマを選んだ理由

自身の子供を授かり、妊娠出産やその後の子育てに関していろいろなサイトやSNSなどをを用いて情報収集を行いました。しかし、ネット上にはいろいろな情報が点在していてどの情報を参考にすればいいのか悩むことが多くありました。
子育てに関する生の声を発信、閲覧することができ、なおかつその投稿を地域別や子供の年齢、性別ごとにソートすることができる機能があれば、より自分に合った情報を見つけやすくなり、私のように子育てに悩む子育て世代の助けになるのではないかと考え、今回のテーマにしました。

### ターゲットユーザ
- 自分の住んでいる地域の子育てイベントを知りたいと考えている人
- 数ある情報の中から、自分の環境にあった子育ての情報を知りたいと考えている人
- 自分の持つ子育ての情報、考えを発信したいと考えている人

### 主な利用シーン
- 自分が知った子育てに関する情報を発信したい時
- 子育てに関する情報の収集がしたい時
- 地域の子育てイベントについて知りたい時
